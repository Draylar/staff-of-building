# Staff of Building

Staff of Building adds in new ~~wands~~ staffs which can be used to speed up building. 
It is built for the 1.16 snapshots on Fabric.
There is a staff for each material, as shown below:

##### Wooden
Radius: 1 

Durability: 100

![](resources/wood.png)

##### Stone
Radius: 2

Durability: 250

![](resources/stone.png)

##### Iron
Radius: 3

Durability: 500

![](resources/iron.png)

##### Golden
Radius: 4

Durability: 150

![](resources/gold.png)

##### Diamond
Radius: 4

Durability: 1000

![](resources/diamond.png)

##### Netherite
Radius: 5

Durability: 1250

![](resources/netherite.png)
